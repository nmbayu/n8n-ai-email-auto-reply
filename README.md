# n8n-ai-email-auto-reply
AI Email Auto Reply built with n8n, Google Gemini and Gmail API
## 📌 Project Overview

This project demonstrates how to build an **AI-powered Email Auto Reply System** using **n8n**, **Google Gemini AI**, and the **Gmail API**.

The workflow receives an email, sends the content to Google Gemini, generates a professional response, and automatically sends the reply through Gmail.

This project is designed as a portfolio example for AI Automation, Workflow Automation, and No-Code/Low-Code development.

---

## 🚀 Features

- 🤖 AI-generated email responses using Google Gemini
- 📧 Gmail API integration
- 🔐 OAuth2 authentication
- ⚡ Workflow automation with n8n
- 🐳 Docker deployment
- 🧩 Easy to customize prompts
- 📂 Exportable n8n workflow
